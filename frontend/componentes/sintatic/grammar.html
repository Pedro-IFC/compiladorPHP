Visualização da Gramática

<Programa>
::= MAIN AP FP AC <ListaComandos> FC | î;
<Declaracao>
::= <Tipo> ID PV;
<Atribuicao>
::= ID EQ <Expressao> PV;
<ChamadaFuncao>
::= ID AP <ListaArgumentos> FP PV;
<ControleFluxo>
::= SE AP <Expressao> FP AC <ListaComandos> FC SENAO AC <ListaComandos> FC | ENQUANTO AP <Expressao> FP AC <ListaComandos> FC | PARA AP ID EQ <Expressao> PV <Expressao> PV <Expressao> FP AC <ListaComandos> FC;
<Imprime>
::= IMPRIMA AP <ArgumentoImpressao> FP PV;
<ArgumentoImpressao>
::= ID | StrLiteral | AS ID AS;
<Expressao>
::= <Termo> <ExpressaoRest> | <ExpressaoLogica>;
<Termo>
::= <Fator> <TermoRest>;
<Fator>
::= ID | CONSTANTESINTEIRAS | CONSTANTESPONTOFLUTUANTE | AP <Expressao> FP;
<Tipo>
::= INT | CHAR | FLOAT;
<ListaArgumentos>
::= <Expressao> <ListaArgumentosRest> | î;
<ListaArgumentosRest>
::= VIR <Expressao> <ListaArgumentosRest> | î;
